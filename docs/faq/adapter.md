# 适配器相关

## OneBot

### 消息发送失败，账号可能被风控

如果你刚开始使用 go-cqhttp，建议挂机 3-7 天，即可解除风控。

### 未连接到 go-cqhttp 子进程

请尝试重新下载 gocqhttp 插件。

### 选择登陆方式的按钮存在但点击无效

已知问题：<https://github.com/koishijs/koishi-plugin-gocqhttp/issues/7>
